# FreeWay China - Лендинг компании

Профессиональный лендинг для компании "FreeWay China", специализирующейся на международной логистике из Китая.

## 🚀 Особенности

- **Современный дизайн** с адаптивной версткой
- **Оптимизированная производительность** (быстрая загрузка)
- **Мобильная адаптивность** (работает на всех устройствах)
- **Интерактивные элементы** (формы, анимации)
- **SEO-оптимизация** (мета-теги, семантическая разметка)

## 🎨 Технологии

## 🚀 Быстрый запуск

### Локально
Просто откройте `index.html` в браузере.

### Через Python сервер
```bash
python3 -m http.server 8000
```

### Через Node.js
```bash
npx http-server -p 8000
```

## 🔧 Настройка

### Изменение контактов
Отредактируйте в `index.html`:
- Номера телефонов
- Email адреса
- Социальные сети
- Адреса

### Изменение цветовой схемы
Измените CSS-переменные в начале `index.html`:
```css
:root {
    --primary: #2563eb;
    --primary-dark: #1d4ed8;
    --secondary: #0ea5e9;
    --accent: #f59e0b;
    --dark: #1e293b;
    --light: #f8fafc;
}
```
