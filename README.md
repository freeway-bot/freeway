## 📋 Концепции

### 🥇 **Concept Premium** — Рекомендуемый
**Премиум корпоративная логистика**
- Темная тема с золотыми акцентами (#0A2540 + #D4AF37)
- Минималистичные анимации при скролле
- Фокус на доверительных элементах и статистике
- Идеально для B2B и корпоративных клиентов

### 🥈 **Concept Startup** — Технологичная платформа
**Тех-стартап логистическая платформа**
- Светлая тема с градиентами (#3B82F6 → #8B5CF6)
- Интерактивная SVG карта маршрутов Китай→Россия
- Современные иллюстрации и tech-эстетика
- Для технологичных компаний и стартапов

### 🥉 **Concept Minimal** — Минималистичная B2B
**Минималистичная современная B2B логистика**
- Монохромная палитра с синим акцентом
- Максимальная читаемость и whitespace
- Фокус на контенте и ясности
- Для клиентов ценящих эффективность

## 🛠 Технологии

### Использовано:
- **HTML5** — семантическая разметка
- **Tailwind CSS** — утилитарные CSS классы
- **Vanilla JavaScript** — нативные анимации
- **Font Awesome** — векторные иконки
- **Google Fonts** — оптимизированная типографика

### Для продакшена рекомендуется:
- **Next.js 14** — для SSR и SEO
- **Framer Motion** — для сложных анимаций
- **GSAP** — для продвинутых timeline анимаций
- **Three.js** — для 3D элементов карты

## 🔧 Настройка и кастомизация

### Для изменения контента:
1. **Текст:** Редактируйте HTML файлы
2. **Цвета:** Измените CSS переменные в `<style>` секции
3. **Анимации:** Настройте параметры в JavaScript файлах
4. **Изображения:** Замените файлы в папке `assets/`

### Для добавления аналитики:
```html
<!-- Google Analytics -->
<script async src="https://www.googletagmanager.com/gtag/js?id=GA_MEASUREMENT_ID"></script>
<script>
  window.dataLayer = window.dataLayer || [];
  function gtag(){dataLayer.push(arguments);}
  gtag('js', new Date());
  gtag('config', 'GA_MEASUREMENT_ID');
</script>
```
