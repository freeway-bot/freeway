# China-Russia Logistics Landing Pages

Мировые лендинги для логистической компании Китай → Россия с фокусом на дизайнерское совершенство, сильную маркетинговую структуру и современные анимации.

## 📋 Концепции

Проект включает 6 различных концепций лендингов:

### 1. **Concept Premium** — Премиум корпоративная логистика
- Темная тема с золотыми акцентами
- Крупная типографика, минималистичный дизайн
- Фокус на доверительных элементах и статистике
- Сдержанные анимации для бизнес-аудитории

### 2. **Concept Startup** — Тех-стартап логистическая платформа (в разработке)
- Светлая тема с яркими акцентами
- Интерактивные элементы и современные иллюстрации
- Анимированные диаграммы и визуализации данных
- Динамичные переходы

### 3. **Concept Minimal** — Минималистичная современная B2B логистика (в разработке)
- Монохромная палитра с акцентным цветом
- Максимальная читаемость и ясность
- Структурированный контент, сеточные макеты
- Субтильные микро-интеракции

### 4. **Concept Animated** — Анимированный интерактивный опыт (в разработке)
- Параллакс скроллинг и 3D трансформации
- Интерактивная карта маршрутов Китай→Россия
- Иммерсивные анимации процессов
- GSAP для сложных анимаций

### 5. **Concept Sales** — Высококонверсионный маркетинговый лендинг (в разработке)
- Множество призывов к действию
- Формы на видных местах
- Социальное доказательство и отзывы
- Ограниченные предложения и таймеры

### 6. **Concept Futuristic** — Смелый футуристический бренд (в разработке)
- Неоновые цвета и глитч-эффекты
- Геометрические паттерны и абстрактные формы
- Экспериментальные анимации
- Cyberpunk эстетика

## 🚀 Запуск проекта

### Для Concept Premium:

1. Перейдите в папку концепции:
```bash
cd concept-premium
```

2. Откройте `index.html` в браузере:
- Двойной клик по файлу
- Или через терминал: `open index.html`

3. Для разработки с live reload (рекомендуется):
```bash
# Установите live-server если нет
npm install -g live-server

# Запустите сервер
live-server --port=3000
```

## 🎨 Дизайн-система Concept Premium

### Цветовая палитра:
- **Primary Dark:** `#0A2540` (темно-синий)
- **Primary Blue:** `#0066FF` (акцентный синий)
- **Accent Gold:** `#D4AF37` (золотой градиент)
- **Light BG:** `#F8FAFC` (светлый фон)
- **Text Dark:** `#1E293B` (основной текст)
- **Text Light:** `#64748B` (вторичный текст)

### Типографика:
- **Заголовки:** Manrope (700-800 weight)
- **Основной текст:** Inter (400-500 weight)
- **Размеры:** 
  - Hero заголовок: 5-7rem (десктоп), 3-4rem (мобильный)
  - Секционные заголовки: 3-4rem
  - Параграфы: 1.125-1.25rem

### Анимации:
1. **Scroll animations** — появление элементов при прокрутке
2. **Hover effects** — микро-интеракции при наведении
3. **Counter animations** — анимация статистики
4. **Parallax** — эффекты глубины (опционально)
5. **Process visualization** — анимация этапов работы

## 🔧 Настройка и кастомизация

### Для изменения контента:
1. **Текст:** Редактируйте HTML файлы
2. **Цвета:** Измените CSS переменные в `<style>` секции
3. **Анимации:** Настройте параметры в `script.js`
4. **Изображения:** Замените файлы в папке `assets/`

### Для добавления новых секций:
1. Создайте HTML структуру
2. Добавьте классы Tailwind
3. Зарегистрируйте анимации в `script.js`
4. Протестируйте на всех устройствах
