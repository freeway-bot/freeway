# Отчет о вдохновении для лендингов логистики Китай → Россия

## Анализ современных высококачественных сайтов

### 1. Stripe
**Ключевые особенности:**
- Чистый минималистичный дизайн с акцентом на типографику
- Плавные градиенты и мягкие тени
- Микро-анимации при наведении
- Четкая визуальная иерархия
- Темная тема с акцентными цветами

**Идеи для заимствования:**
- Использование градиентов для создания глубины
- Анимации при скролле для рассказывания истории
- Четкие призывы к действию

### 2. Linear
**Ключевые особенности:**
- Ультра-минималистичный интерфейс
- Прецизионная типографика
- Субтильные анимации
- Фокус на контенте
- Монохромная палитра с акцентами

**Идеи для заимствования:**
- Минималистичный подход для B2B аудитории
- Фокус на читаемость и ясность
- Сдержанные анимации

### 3. Vercel
**Ключевые особенности:**
- Смелые геометрические формы
- Динамичные градиенты
- Интерактивные элементы
- Современные иллюстрации
- Быстрые переходы

**Идеи для заимствования:**
- Использование геометрических паттернов
- Динамичные визуальные эффекты
- Интерактивные демонстрации

### 4. Framer
**Ключевые особенности:**
- Экспериментальные анимации
- Параллакс эффекты
- 3D трансформации
- Плавные переходы между секциями
- Иммерсивный опыт

**Идеи для заимствования:**
- Сложные scroll-анимации
- Параллакс для создания глубины
- Интерактивные элементы

### 5. Notion
**Ключевые особенности:**
- Чистый, организованный дизайн
- Система сеток
- Консистентные spacing
- Минимальные украшения
- Фокус на функциональность

**Идеи для заимствования:**
- Структурированный контент
- Четкая организация информации
- Консистентные отступы

### 6. Flexport (логистическая компания)
**Ключевые особенности:**
- Карты и визуализации маршрутов
- Данные и статистика
- Процессные диаграммы
- Кейс-стади
- Доверительные элементы

**Идеи для заимствования:**
- Визуализация логистических маршрутов
- Отображение данных и метрик
- Демонстрация процесса

## Паттерны макетов для логистических сайтов

### 1. Hero секция
- Крупный заголовок с ценностным предложением
- Подзаголовок с уточнением
- Призыв к действию (CTA)
- Фоновое изображение/видео/иллюстрация
- Индикатор скролла

### 2. Секция услуг
- Иконки или иллюстрации для каждой услуги
- Краткое описание
- Ссылки на детальные страницы
- Grid layout (3-4 колонки)

### 3. Процесс работы
- Визуальная временная шкала
- Нумерованные шаги
- Иконки для каждого этапа
- Описание процесса

### 4. Доверительные элементы
- Логотипы партнеров
- Статистика (количество доставок, стран и т.д.)
- Отзывы клиентов
- Сертификаты
- Кейс-стади

### 5. Лид-генерация
- Форма запроса котировки
- Контактная информация
- WhatsApp/Telegram кнопки
- Чат-бот

## Типографика

### Тренды 2025-2026:
1. **Крупные заголовки** - 64-96px для hero секций
2. **Variable fonts** - адаптивная толщина
3. **Моноширинные шрифты** для технических деталей
4. **Контрастные комбинации** - sans-serif + serif
5. **Анимированная типографика** - появление букв

### Рекомендуемые шрифты:
- **Inter** - чистый, современный
- **SF Pro Display** - Apple-стиль
- **Manrope** - геометрический
- **Clash Display** - смелый, выразительный
- **JetBrains Mono** для технических деталей

## Цветовые палитры

### Для логистики:
1. **Синие тона** - доверие, профессионализм
2. **Оранжевые акценты** - энергия, скорость
3. **Нейтральные фоны** - белый, светло-серый
4. **Темные темы** - для премиум ощущения
5. **Градиенты** - синий → фиолетовый, оранжевый → красный
